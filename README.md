# Class Visualizer
This is a source code of [Class Visualizer](http://class-visualizer.net/).

## Build
```
mvn clean package
```

## Run
```
mvn exec:exec
```

## Dependencies
- JDK 1.8 / 11+
- Maven 3+ - for build

## IDE
NetBeans 8.1+ is required for UI development. For everything else - any IDE can be used.
